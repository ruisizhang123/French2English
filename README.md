# French2English

##### Dependencies

 * Python >= 3.6
 * torch >= 0.4.1

##### Dataset 

  * [French2English](http://www.statmt.org/europarl/)

##### Feature

  * Translate French to English
  * Visualize Attention 
  
##### Usage

  * Run `python translate.py -h`  to get help information
  * Run `python translate.py \ --sentence "French Sentence to Translate"` to translate sentence.
